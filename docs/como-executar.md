# Como executar

Dentro da pasta do projeto, execute os comandos abaixo:

```bash
npm install
npm run dev
```

O projeto estará disponível em [http://localhost:5173](http://localhost:5173).

!!! tip "Dica"
    Certifique-se de ter o Node.js 24 ou superior e o npm 11.7.0 ou superior instalados antes de executar o projeto.
